# Day 1 - Environment Setup (2 hrs)

Goal: Your machine is ready to build

## 0:00-0:20
- Install Python 3.11+, pip, virtualenv.
- Create project folder: ai-backend/

## 0:20-0:50
- Install FastAPI, Uvicorn, Pydantic.
- Run: uvicorn main:app --reload with Hello World.

## 0:50-1:20
- Install PostgreSQL locally + pgAdmin or DBeaver.
- Create DB: aidb

## 1:20-2:00
- Push project to GitHub.
- Write a proper README.
- First commit done.
